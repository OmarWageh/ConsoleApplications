# Task Tracker Console App

A simple Task Tracker application built using C# Console Application.

## Features

* Add Tasks
* View Tasks
* Update Tasks
* Delete Tasks
* Limit the number of tasks
* Console menu system

## Technologies Used

* C#
* .NET Console Application
* Generic List Collection

## How It Works

1. User enters the maximum number of tasks.
2. User chooses options from the menu:

   * Add Task
   * View Tasks
   * Delete Task
   * Update Task
   * Exit Application

## Project Structure

* `AddTask()` → Adds a new task
* `ViewTasks()` → Displays all tasks
* `DeleteTask()` → Removes a task by index
* `UpdateTask()` → Updates task name by index
